## Project Overview

This project performs **Exploratory Data Analysis (EDA)** on the IMDb Top 1000 Movies dataset to gain insights into movie genres, ratings, popularity, and trends. The analysis includes data cleaning, visualization, and sentiment scoring for movie overviews.
The goal is to better understand how different genres relate to IMDb ratings and which categories of movies resonate with audiences.

---

## Objectives

- Analyze genre distribution among the top 1000 IMDb movies  
- Compare average ratings across genres  
- Identify trends in movie popularity  
- Perform sentiment analysis on movie descriptions
- Perform movie recommendations
- Perform Data Preprocessing,Feature Engineering, Data Visualization
- Perform supervised and unsupervised learning
